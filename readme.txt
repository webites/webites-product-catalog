You can show your products to customers for free! Great plugin with elastic options to display catalog, single products with meta data, images, description and available status.

== Description ==
 
Add your products to plugin and give your customers better research what do you offer.

You can add to product:
* colors
* dimensions
* SKU
* series
* type
* price

Every single product have own image, title and description. You can add that in administration panel and it is very simple.

Taxonomies of products is: categories and tags.

Your can enable/disable display of that:
* image
* metadata
* categories and tags
of products

In archive of products (products catalog) items can be displays on two ways: 
* grid
* simple method (all items have 100% width)

Simple product page can look like:
* e-commerce product page
* default plugin product page

You can set product from our plugin as product in SEO plugin to better indexing.

== Installation ==
 
This section describes how to install the plugin and get it working.
 
Simplest:

1. Run your Wordpress
2. Plugins -> Add New
3. Search Plugin: **weBites Product Catalog**
4. Click **Install Now**
5. After Install click **Activate**

Manual:

1. Download .zip file
2. Plugins -> Add New
3. Upload Plugin 
4. After Install click **Activate**

== Frequently Asked Questions ==
 
= Where Plugin displays products =
 
The plugin displays products where you set them. On menu settings you can enable display of categories, tags or single products.

= Can I modify plugin =

Yes, but after update you will lost all changes. Better use hooks, native functions and CSS styles. If you want modify the Plugin to your needs you can contact with us on [weBites website](https://webites.pl "weBites website")!
